Dans le cadre de son activité, Planet On Stage est amené à collecter et à traiter des informations dont certaines sont qualifiées de "données personnelles".

notes: >- alt: >- credit: >- - kind: block template: chapter title: >- Données personnelles slug: >- donnees-personnelles ranks: self: 2 data: layout: no_background text: >-

Sur le site web de Planet On Stage, il n'y a qu'un type de données susceptibles d’être recueillies : des données d'identité. Ces données sont celles que vous nous transmettez directement par contact direct par email.

notes: >- alt: >- credit: >- - kind: block template: chapter title: >- Utilisation des données slug: >- utilisation-des-donnees ranks: self: 2 data: layout: no_background text: >-

Les données que vous nous transmettez directement sont utilisées dans le but de vous re-contacter et/ou dans le cadre de la demande que vous nous faites. Ces données sont utilisées par Planet On Stage responsable du traitement des données, et ne seront jamais cédées à un tiers ni utilisées à d’autres fins que celles détaillées ci-dessus.

notes: >- alt: >- credit: >- - kind: block template: chapter title: >- Base légale slug: >- base-legale ranks: self: 2 data: layout: no_background text: >-

Les données personnelles ne sont collectées qu’après consentement obligatoire de l’utilisateur. Ce consentement est valablement recueilli cases à cocher, libre, clair et sans équivoque.

notes: >- alt: >- credit: >- - kind: block template: chapter title: >- Cookie slug: >- cookie ranks: self: 2 data: layout: no_background text: >-

Nous n'utilisons pas de cookies.

notes: >- alt: >- credit: >- - kind: block template: chapter title: >- Vos droits concernant les données personnelles slug: >- vos-droits-concernant-les-donnees-personnelles ranks: self: 2 data: layout: no_background text: >-

Vous avez le droit de consultation, demande de modification ou d’effacement sur l’ensemble de vos données personnelles. Vous pouvez également retirer votre consentement au traitement de vos données.
